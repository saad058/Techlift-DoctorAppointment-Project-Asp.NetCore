Dropzone

http://www.dropzonejs.com/

Dropzone.js is a light weight JavaScript library that turns an HTML element into a dropzone. This means that a user can drag and drop a file onto it, and the file gets uploaded to the server via AJAX.

Examples

Drop files here or click to upload (This is just a demo dropzone. Selected files are not actually uploaded.)

Usages

<link rel="stylesheet" href="assets/plugins/dropzone/dropzone.css">
<script src="assets/plugins/dropzone.js/dropzone.js"></script>
<form action="/upload" class="dropzone needsclick" id="dropzone-demo">
	 <div class="dz-message needsclick">
		Drop files here or click to upload
		<span class="note needsclick">(This is just a demo dropzone. Selected files are
		<strong>not</strong> actually uploaded.)</span>
	 </div>
	 <div class="fallback">
		<input name="file" type="file" multiple>
	 </div>
</form>
